# OpenClaw Skills - 求婚策划

本仓库包含用于 OpenClaw 的求婚策划技能。

## proposal-planner
浪漫求婚策划助手，帮助你策划和执行完美的求婚计划。

**功能：**
- 收集求婚基本信息（姓名、相识时间、预算等）
- 场地推荐（私人影院、海边、家里等）
- 惊喜环节设计（保密策略、惊喜方式）
- 求婚词撰写指导（模板、结构、要点）
- 全流程准备清单（时间线规划）

## 使用方法

将 skill 复制到你的 OpenClaw 工作区 skills 目录

然后在 OpenClaw 中使用：
- 帮我策划求婚
- 求婚礼场地推荐
- 求婚词怎么写

## 适用场景
- 想求婚但不知如何开始
- 想给女朋友惊喜
- 需要在特定日子求婚
- 不知道场地怎么选
