## 检查项21：复位信号使用前检查 [S]

### 检查方法
复位信号跨时钟域时，是否先做同步再使用

### 检查结果
❌ **未通过**

### 检查详情
复位信号`res`在模块外部产生，可能来自其他时钟域，但代码中未对`res`进行同步处理就直接使用。

**代码位置**: 第31行
```verilog
always @(posedge clk or negedge res) begin
```

### 参考规范
`Verilog_RTL_Coding_Standards-v1.4.3.md` 第10条

### 建议
如果复位信号可能跨时钟域，应先进行同步处理：
```verilog
reg res_sync1, res_sync2;

always @(posedge clk) begin
    res_sync1 <= res;
    res_sync2 <= res_sync1;
end

always @(posedge clk or negedge res_sync2) begin
    if (~res_sync2) begin
        // 复位逻辑
    end
end
```
