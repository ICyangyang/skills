## 常见问题汇总

### 问题1：非ASCII字符使用
- **文件**: UART_RXer.v, UART_TXer.v
- **位置**: Line 52
- **问题**: 使用中文全角分号（；）
- **修复建议**: 替换为ASCII分号（;）

### 问题2：信号命名建议改进
- **建议**: 复位信号建议使用 `rst_n` 或 `rst_n` 命名（低电平有效加 `_n` 后缀）
- **当前**: 使用 `res` 作为复位信号名

### 问题3：模块例化命名
- **建议**: 例化名建议使用 `u_` 前缀
- **当前**: `UART_RXer UART_RXer(...)` 例化名与模块名相同

## 阶段总结

1. **中文全角字符问题**：代码中混入了中文全角分号，需立即修复
2. **复位信号命名**：建议使用更规范的 `rst_n` 命名
3. **例化名规范**：建议使用 `u_` 前缀区分例化名与模块名
