### 第 25 项：变量声明检查

**参考值**: 否

### 第 26 项：变量声明冲突性检查
**规则级别**: [S]
**检查方法**: 局部变量名和全局变量名是否发生冲突
**参考值**: 否

### 第 27 项：变量赋值检查
**规则级别**: [R]
**检查方法**: 赋值语句中是否使用"X"或是"Z"
**参考值**: 否

## 检查要点

1. **信号声明**: 不应在声明时赋初始值
   ```verilog
   // 错误
   reg [7:0] data_reg = 8'h00;

   // 正确
   reg [7:0] data_reg;
   ```

2. **变量命名冲突**: 检查层次化命名

3. **X和Z的使用**:
   - 避免在赋值语句中使用X或Z
   - X和Z仅在testbench中使用
